# Personal Portfolio Website

## 👩‍💻 About the Project

This repository contains my personal portfolio website which showcases my skills, projects, and experience as a software developer. The portfolio is designed to present my work, technical abilities, and projects in a clean and responsive interface.

The website acts as a central hub where visitors can learn about me, explore my projects, and view the technologies I work with.

## 🚀 Features

* Responsive portfolio design
* Clean and simple user interface
* Project showcase section
* About Me section
* Skills display
